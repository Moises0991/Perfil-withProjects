Paginas & Proyectos

En esta seccion se pretende presentar los proyectos marcados para
la asignatura de Programacion Web asignados por el profesor
Carlos Tamay

CALCULADORA

En este apartado se pretende elaborar una calculadora que cumpla unicamente con la las operacines basicas de suma, resta, multiplicacion y divicion.

AUTOBIOGRAFIA

En este proyecto la Autobiografia viene siendo la pagina en la que nos encontramos y como complemento se adapto para poder presentar todos los proyectos juntos.

CARRITO DE COMPRAS

En esta ultima pagina tienen varios articulos para agregar al carrito ( o quitar), estos tienen nombre, precio e imagen (que cambian aleatoriamente).

Acerca de la Materia

En este 4to cuatrimestre que paso pudimos aprender a como desarrollar Paginas Web implentando el estandar de desarrollo HTML5, lengujes como php para conectar con el servidor y tambien herramientas como bootstrap que nos ayudaron a hacer mas facil el diseño de las paginas.

Mas acerca del Proyecto

Para que este proyecto funcione de forma correcta en el ordenador (de momento) es necesario que se cuente con internet ya que los repositorios se encuentran en la nube y algunos elementos se llaman mendiante ligas.
